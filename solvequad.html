Solving Quadratics

Click START to generate a random quadratic equation to solve.

All answers should be in lowest terms and simplest radical form. That means if you get an answer like:

\(x=\frac{28\pm\sqrt{320}}{12}\)

It is really:

\(x=\frac{28\pm\sqrt{320}}{12}=\frac{28\pm8\sqrt{5}}{12}=\frac{4(7\pm2\sqrt{5})}{4(3)}=\frac{7\pm2\sqrt{5}}{3}\)

And should be typed it as (2+√2)/3 as the solution for \(x_1\) and (2+√2)/3 for \(x_2\). Note that you need the parentheses around the numerator.

If you type "sqrt", it will be replaced with √, and if you type "no", it will be replaced with ∅ to represent no solution. If there is only one solution, put the ∅ as the solution for \(x_2\).

Best Methods Based on Type of Quadratic

  • Standard Form: \(ax^2+bx+c=0\)
    ▹Try factoring or graphing first

    ▹If that doesn't work, use the quadratic formula: \(x=\frac{-b\pm\sqrt{b^2-4ac}}{2a}\)

  • Vertex Form: \(a(x-h)^2+k=0\)
    ▹Use inverse operations
