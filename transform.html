Identifying Transformations

Click the button to generate a function. Then give its shift left/right, shift up/down, vertical stretch/compression, vertical reflection, horizontal stretch/compression, and horizontal reflection.

What the Variables do

  • \(a\) - Vertical stretch if \(|a|>1\), compression if \(|a|<1\), and reflection if \(a<0\)

  • \(b\) - Horizontal compression if \(|b|>1\), stretch if \(|b|<1\), and reflection if \(b<0\)

  • \(h\) - Horizontal shift right if \(h>0\), shift left if \(h<0\)

  • \(k\) - Vertical shift up if \(k>0\), shift down if \(k<0\)
